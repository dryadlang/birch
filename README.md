# Birch

Dryad native integrated development environment.

## Overview
Birch is an IDE for the Dryad programming language, built with Tauri and React.

## Development

This project uses [Tauri](https://tauri.app), [React](https://reactjs.org), and [Vite](https://vitejs.dev).

### Prerequisites
- Rust (latest stable)
- Node.js (LTS)

### Running
```bash
npm install
npm run tauri dev
```
